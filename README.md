# Scriptish

[![Gitter chat](https://badges.gitter.im/scriptish/scriptish.svg)](https://gitter.im/scriptish/scriptish)

## About

Scriptish is a userscript manager for Firefox, forked from Greasemonkey.

## Browsers Supported

* Firefox 21+
* Seamonkey 2.8+
